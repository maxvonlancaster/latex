0.61
  NEW:
   o csquotes.sty compatibility for citations inside \blockquote introduced
        (bug reported by Bernhard Tempel, solution provided by Philipp Lehman).
   o translator names have their own command now: \bibtnf, this makes configuration
        separately from author or editor names possible.
   o 'splitidx' compatibility implemented (requested by Christian Neumann).
   o jurabib supports swedish now (contributed by Anders Dahnielson).
   o New value 'halfstrict' for option 'idem' contributed by Martin Sievers.
        This value supresses the output of idem inside  an bibliographic entry,
        e.g. if a work has the same author and editor.
   o After a feature request from Friedrich Vosberg in d.c.t.t.,
        a new field `noauthorcite' for @commented was introduced. If the value of this field = 1,
        jurabib will use a given shorttitle instead of the (generated or given) shortauthor.
        This might be useful for german law students/lawyers citing big comments (like MüKo), which are
        usually not cited with the names of the editors but with the shorttitle (e.g. 'Müko BGB').
   o ibidem options are now available for the natbib commands too.
   o new option 'nopublisher' with three values:
        - `cite': supresses publisher output in citations.
        - `bib': supresses publisher output in bibliography.
        - `both': supresses publisher output in citations and bibliography.
        (Patch suggested by Seb Wills)
   o \space added in \numberandseries.
   o new option 'bibformat=shorttitle', places given shorttitle after
        author name in the bibliography (does not work with `bibformat=ibidemalt').
   o new command \bibidemhrule, allows modifying of the printed rule
        used by `bibformat=idem'.
   o jurabib-BibTeX styles have a new field `pagestype' to make
        usage of pages field for non-pages possible, i.e. to cite rows
        of a lexicon using `pages=2' and `pagestype={row}' will give
        you exactly `row 2' in both citation and bibliography (suggested by Florian Schmitt).
        If pagestype is omitted, jurabib will assume pages and will print `p.' or `pp.'
   o improved gender tests for 'bibformat=ibidem'.
   o translators are now formatted.
   o new option `pages=ignoreibidem' introduced to print out pages for ibidem citations 
            even if the pages are the same.

   o  new option `annotatorfirstsep', replaces `cofirstsep' option.
   o  new option `annotatorformat', replaces `coauthorformat' options.
   o  new option `super' transforms \cite commands to footnotes,
        suggested by Frank Mittelbach.
   o  new option `config=<configfile without extension>', allows to
        switch between configfiles, suggested by Frank Mittelbach.
   o  new command \lookatfortype, allows to determine a list of entry types
        for which the lookat feature should be used (suggested by David Feest).
   o  \footcitetitleonly introduced, suggested by Frank Mittelbach.
   o  New commands to tune indexing introduced:
        \jbdonotindexeditors (suppress editor-indexing).
        \jbdonotindexauthors (suppress author-indexing).
        \jbdonotindexorganizations (suppress org-indexing).
        \jbindexonlyfirstauthors (to index authors only at first appearence).
        \jbindexonlyfirsteditors (to index editors only at first appearence).
        \jbindexonlyfirstorganizations (to index organizations only at first appearence).

        aren't sorted properly (makeindex-problem).

        \jboyearincitation.

        1. \usepackage option overwrites values given by config files.
        2. option placed in \jurabibsetup (in the preamble) overwrites
           values given by \usepackage.
        3. last \jurabibsetup overwrites values of preceding \jurabibsetup.
    o  Some hooks for multibib.sty compatibility implemented, testfiles
        added for multibib compatibility (jbtestmb.tex) and for
        bibunits compatibility (jbtestbu.tex), thanks to Thorsten Hansen
        for adapting his packages.

        suggested by Frank Mittelbach.

        New commands:
          \formatarticlepages[<separator>]{<page left delim>}{<page right delim>}
             This command allows you to print out a (addtional) startpage
             of a given pagerange, before the ``normal'' page(range), given
             by the optional argument of \cite.
             Assumed, you used \formatarticlepages[: ]{(}{)}
             in the preamble, you typed `pages = {40--60}' in the .bib
             file and you wrote \cite[45 and 47]{<key>} in the text,
             the output will look like that:
             ..., 40: (45 and 47)
             or with option `pages=format':
             ..., p. 40: (pp. 45 and 47)
