Welcome to the mapcodes system!

This file contains the distribution guide for the above-named version
of the mapcodes package for the support of 8-bit input file in a
variety of encodings. This package was written and is maintained by
Michael Piotrowski.

You need LaTeX2e for this package.

This distribution is described in the files ending with .txt.  You
should read install.txt before starting to install mapcodes.

 - 00readme.txt is the file you are reading.

 - install.txt  describes how to install mapcodes.

 - manifest.txt lists all the files in the mapcodes distribution.

You are not allowed to change the files in this distribution.
